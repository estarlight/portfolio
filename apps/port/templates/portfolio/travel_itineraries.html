Travel Itineraries

Overview

Travel Itineraries can be described as a “Pinterest for travel guides.” Users can register for an account, and create itineraries that document their own travels, with guides and tips for future travelers.

Users can also search the database by location for itineraries shared by other users. They can view these itineraries for inspiration for future destinations, or save them to “My Saved Itineraries” to keep on file while planning their next trip.

Language

Python

Technologies

Django

SQLite

Bootstrap

jQuery

AJAX

My inspiration for creating this web application was my own penchant for travel. I am a planner, so I’ll be researching destinations months in advance to create a travel itinerary for my trip.

The only hangup I had was that my research was spread across so many different websites, from travel sites like TripAdvisor, to tons of personal travel blogs.

I wanted to create a site where all tips, itineraries and travel guides could be stored and searched for in one place, so that travel planning could be simpler and more enjoyable!
