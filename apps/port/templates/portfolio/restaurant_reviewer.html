Let's Eat!

Overview

A simple web application that allows users to add, rate, and review restaurants.

Built with the MEAN stack, this application incorporates the front-end framework Angular with a MongoDB database. Bootstrap was also used to design the front-end website.

Users can add, edit, or delete restaurants, read existing reviews, and add reviews for a particular restaurant.
